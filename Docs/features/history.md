# 历史记录

::: warning 🚧 计划中
此功能正在规划中，预计将在 **v0.4.0** 版本推出。
:::

## 功能概述

本地存储最近的语音转写记录，方便回顾和复用。

## 计划功能

- **自动保存** — 每次转写自动记录
- **快速复制** — 一键复制历史内容
- **搜索过滤** — 按关键词查找
- **数量限制** — 自定义保留条数
- **隐私保护** — 数据仅存储在本地

## 存储位置

```
~/Library/Application Support/EtherType/History/
```

## 隐私说明

- ✅ 完全本地存储
- ✅ 不会上传云端
- ✅ 可随时清除
