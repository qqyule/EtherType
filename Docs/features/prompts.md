# 自定义提示词

::: warning 🚧 计划中
此功能正在规划中，预计将在 **v0.3.0** 版本推出。
:::

## 功能概述

允许用户自定义后处理提示词，针对不同场景自动优化文本输出格式。

## 计划功能

- **预设模板** — 内置常用场景模板
- **自定义创建** — 编写个性化提示词
- **快速切换** — 一键切换不同场景
- **导入/导出** — 分享你的提示词配置

## 预设场景

| 场景 | 描述 |
|------|------|
| 📝 会议记录 | 自动分段、提取要点 |
| ✉️ 邮件撰写 | 正式格式、礼貌用语 |
| 💻 代码注释 | 符合代码规范的注释格式 |
| 📱 社交媒体 | 轻松活泼的语气 |
| 📄 技术文档 | 严谨专业的表达 |

## 工作原理

```
语音识别结果 + 场景提示词 → Ollama → 优化输出
```

## 依赖功能

此功能依赖 [Ollama 后处理](/features/ollama) 功能。
