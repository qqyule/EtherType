# 实时转写

::: warning 🚧 计划中
此功能正在规划中，预计将在 **v1.0.0** 版本推出。
:::

## 功能概述

边说边显示文字，无需等待录音结束，实现流畅的实时输入体验。

## 计划功能

- **流式输出** — 说话时即时显示文字
- **动态更新** — 随着上下文自动修正
- **低延迟** — 优化推理速度
- **视觉反馈** — 实时 HUD 显示

## 与当前模式对比

| 特性 | 当前模式 | 实时转写 |
|------|----------|----------|
| 输出时机 | 松开按键后 | 说话时即时 |
| 修正能力 | 一次性输出 | 动态修正 |
| 适合场景 | 短句输入 | 长文本听写 |

## 技术挑战

- 需要流式推理支持
- 要平衡延迟和精度
- 内存占用优化
