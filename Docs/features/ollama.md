# Ollama 后处理

::: warning 🚧 计划中
此功能正在规划中，预计将在 **v0.3.0** 版本推出。
:::

## 功能概述

集成本地 Ollama LLM，对语音识别结果进行智能后处理，提升文本质量。

## 计划功能

- **标点优化** — 自动添加正确的标点符号
- **格式化** — 根据上下文自动分段、缩进
- **纠错** — 修正语音识别的常见错误
- **模型可选** — 支持 Ollama 中的任意模型

## 使用场景

| 场景 | 效果 |
|------|------|
| 会议记录 | 自动分段、添加要点标记 |
| 邮件撰写 | 添加正式格式、礼貌用语 |
| 代码注释 | 格式化为标准注释格式 |

## 技术架构

```
语音 → WhisperKit → 文本 → Ollama API → 优化文本 → 输出
```

## 先决条件

- 需要本地安装 [Ollama](https://ollama.ai/)
- 至少下载一个语言模型（如 `llama2` 或 `qwen`）
