# 快速开始

在几分钟内开始使用 EtherType。

## 下载安装

1. 前往 [Releases 页面](https://github.com/yourusername/EtherType/releases)
2. 下载最新版本的 `.dmg` 文件
3. 打开 `.dmg`，将 EtherType 拖入 `Applications` 文件夹

## 首次启动

1. 打开 EtherType，应用将常驻菜单栏（无 Dock 图标）
2. 首次启动会弹出权限请求：
   - **麦克风权限** — 用于录制语音
   - **辅助功能权限** — 用于将文字输入到其他应用
3. 授权后，应用会自动下载语音识别模型（约 500MB）

## 开始使用

1. 按住 `Option + Space`（默认快捷键）
2. 对着麦克风说话
3. 松开按键，语音将自动转为文字并输入到当前光标位置

::: tip 提示
EtherType 会在屏幕中央显示一个动态声波 HUD，表示正在聆听。
:::
