# 快捷键配置

EtherType 使用全局快捷键触发语音输入，你可以自定义为任意组合键。

## 默认快捷键

| 功能 | 默认快捷键 |
|------|-----------|
| 按住录音 | `Option + Space` |

## 自定义快捷键

1. 点击菜单栏中的 EtherType 图标
2. 选择「设置」
3. 在「快捷键」选项卡中，点击录制区域
4. 按下你想要的快捷键组合
5. 完成后自动保存

::: tip 推荐的快捷键
- `Option + Space` — 默认，不会与大多数应用冲突
- `Control + Space` — 如果你没有使用输入法切换
- `Fn + F5` — 利用功能键
:::

## 使用方式

EtherType 采用「对讲机」模式：

1. **按住**快捷键开始录音
2. 屏幕中央出现声波 HUD
3. **松开**快捷键结束录音
4. 文字自动输入到当前光标位置

::: warning 注意
快捷键需要在授予「辅助功能」权限后才能生效。
:::

## 冲突处理

如果你设置的快捷键与其他应用冲突：

1. 检查系统设置中的快捷键配置
2. 考虑使用带 `Fn` 修饰符的组合
3. 某些应用（如 IDE）可能会拦截快捷键，建议在这些应用中使用不同的组合
