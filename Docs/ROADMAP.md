## 🤝 开源社区

### 欢迎贡献

- 提交 Issue 报告 Bug 或建议功能
- 提交 PR 参与开发
- 分享使用体验

### 贡献指南

```bash
# Fork & Clone
git clone https://github.com/qqyule/EtherType.git

# 创建功能分支
git checkout -b feature/your-feature

# 开发 & 测试
swift build
swift test

# 提交 PR
```

### 代码规范

- Swift 6.0 语法
- SwiftUI 优先
- 中文注释 (JSDoc 风格)
- 遵循 MVVM 架构

## 💡 设计原则

1. **隐私优先** — 数据永不离开本地
2. **极简至上** — 做减法，不做加法
3. **原生体验** — 遵循 macOS 设计语言
4. **开源透明** — 代码即文档
