# 常见问题

## 安装与启动

### 为什么首次启动很慢？

首次启动时，EtherType 需要下载语音识别模型（约 500MB）。请确保网络畅通，耐心等待下载完成。

### 提示"无法验证开发者"怎么办？

由于应用未经过 Apple 公证，首次打开时需要：

1. 右键点击应用图标
2. 选择「打开」
3. 在弹出的对话框中再次点击「打开」

### 菜单栏看不到图标？

EtherType 是一个菜单栏应用，不会出现在 Dock 中。请查看菜单栏右侧区域。

如果菜单栏图标过多，可以按住 `Command` 拖动图标调整位置。

## 权限问题

### 辅助功能权限如何授予？

1. 打开「系统设置」
2. 进入「隐私与安全性」→「辅助功能」
3. 点击锁图标解锁
4. 点击「+」添加 EtherType
5. 确保开关已打开

### 为什么需要麦克风权限？

EtherType 需要录制你的声音才能进行语音识别。没有麦克风权限，应用无法工作。

---

## 使用问题

### 快捷键没反应？

1. 检查辅助功能权限是否已授予
2. 检查快捷键是否与其他应用冲突
3. 尝试重新设置快捷键

### 识别不准确？

1. 尝试切换到 Large V3 模型
2. 确保麦克风质量良好
3. 减少环境噪音
4. 说话速度适中，吐字清晰

### 文字无法输入到某些应用？

某些应用（如终端、安全软件）可能会阻止外部输入。这是系统安全机制，无法绕过。

## 其他

### 如何卸载？

1. 退出 EtherType（菜单栏 → 退出）
2. 将 Applications 中的 EtherType 移到废纸篓
3. 可选：删除 `~/Library/Application Support/EtherType/`

### 如何反馈问题？

请在 [GitHub Issues](https://github.com/qqyule/EtherType/issues) 提交问题。

提交 Issue 时，请尽量包含：
- macOS 版本
- 问题描述

- 相关截图
